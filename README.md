# param_ecm

Параметры **ECM** сохраняются в следующих файлах:

- **1RC модель:**  
  `plsDS_soc455055_fr2_1RC_R0free_main.csv`

- **2RC модель:**  
  `plsDS_soc455055_fr2_2RC_R0free_main.csv`

В **последнем столбце** каждой таблицы указано имя файла, в котором находятся данные соответствующего *токового импульса*, по которому рассчитаны параметры ECM.

---

## Источники импульсов

Данные импульсов, используемых для идентификации параметров, лежат в отдельных папках:

- Для **1RC** модели:  
  `plsDS_soc455055_fr2_1RC_R0free_pls_tables`

- Для **2RC** модели:  
  `plsDS_soc455055_fr2_2RC_R0free_pls_tables`

Каждый файл в этих директориях представляет собой таблицу импульса, из которой были получены соответствующие параметры ECM.
